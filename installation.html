Guide to install: Git and GitHub

For Git, users should go to https://git-scm.com/downloads to install Git to their personal devices. When the file is successfully downloaded, users can open the file and go through the specific instructions to install Git to the device. Git is available for MacOS, Windows and Linux/Unix.

In terms of GitHub, users will need to go to https://github.com and register for an account. If you are a student, you can apply for the educational package that unlocks more features of GitHub.
